# Library App

This is a Node.js CRUD application connected to a MySQL database called **LIBRARY**.

## Features
- View all books
- View a single book by ID
- Add a new book
- Update book details
- Delete a book

## Endpoints
- `GET /book` – Display all books  
- `GET /book/:id` – Display a single book by ID  
- `POST /book` – Add a new book  
- `PUT /book/:id` – Update a book  
- `DELETE /book/:id` – Delete a book
